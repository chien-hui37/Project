## **概述**
### 技術
- **Frontend:** React.js
  - useContext, useForm, useEffect  
- **Backend:** Node.js
  - JWT Token
- **API:** Axios
- **Database:** AWS RDS (MySQL)
- Other: RWD Design, Bootstrap5

### 目的
- **一站式服務**
- **用戶友好設計**
- **專業、可靠的訊息**

### 貢獻
- **預約照服員**
- **長照據點地圖**
